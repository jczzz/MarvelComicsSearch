# MarvelComicsSearch

 A hybrid mobile application that displays a searchable list of Marvel superheroes/characters along with information about the characters and comics.

## Version
```sh
node : v8.9.3
```
```sh
ionic: v3.20.0
```





## Installation

Install  dependencies
```sh
npm install
```

## Run the app on the browser

```sh
ionic serve
```
